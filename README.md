# econ270c_reproduction
Revised reproduction package for Asher and Novosad (AER, 2020): Rural Roads and Local Economic Development.

Note that some data files are not included because of the data limit in Github. One needs to download the 'data' folder from the ICPSR repository of this paper. https://www.openicpsr.org/openicpsr/project/109703/version/V2/view
